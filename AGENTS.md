# AGENTS.md — Правила и стандарты проекта Litestar

## Технологический стек
- Python 3.10+
- Litestar (последняя стабильная версия)
- Uvicorn (ASGI сервер)
- Jinja2 (серверный рендеринг HTML)
- CSS: Tailwind CSS через CDN (без сборки)
- Валидация: Pydantic v2 (встроена в Litestar)

## Жёсткие правила кода
1. **Типизация:** Все функции, параметры и возвращаемые значения должны иметь аннотации типов. Litestar использует их для DI и валидации.
2. **Асинхронность:** Все route-handlers объявлять как `async def`.
3. **Структура:** Маршруты группировать в контроллеры (`class MyController(Controller)`). Не писать всё в одном файле.
4. **Шаблоны:** HTML рендерить через `Template(name="...")`. Не возвращать сырые HTML-строки из контроллеров.
5. **Статика:** CSS/JS подключать через `StaticFilesConfig` или `create_static_files_router`.
6. **Ошибки:** Использовать кастомные exception handlers. Не оставлять tracebacks в продакшене.
7. **Стиль:** PEP 8, отступы 4 пробела, понятные имена переменных.

## Запреты
- ❌ Не использовать глобальные переменные для состояния приложения.
-  Не хардкодить пути к шаблонам/статике.
- ❌ Не игнорировать type hints.
